# Simple Contact Us Form Management Plugin

This is octobercms plugin that we use to show contact us form for visitor. 

To use the plugins:

```
$ cd octobercmsproject/plugins
$ mkdir yfktn
$ cd yfktn
$ git clone https://github.com/yfktn/contactus-plugin.git contactus
```
Beware if this is still in the early stage of development.
